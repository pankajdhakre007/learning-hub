How to extend this package

Add each new static page into the docs folder and reuse assets/css/site.css. If you need page-specific client logic, add it under assets/js/, then include it with a script tag.

  • Keep page structure consistent with the shared layout classes.
  • Reuse the same .page-shell, .nav-card, and .button patterns.
  • Place animation scripts in assets/js/ to avoid duplication.
